# DevSolve 사용자 시나리오

## 프로젝트 개요

DevSolve는 1:1 개발자 멘토링 플랫폼입니다. 멘토(개발자)와 멘티(학습자)를 연결하여 맞춤형 과외 서비스를 제공합니다.

---

## 전체 페이지 구조 (17개 라우트)

| 경로 | 페이지 | 역할 | 설명 |
|------|--------|------|------|
| `/` | HomePage | 공통 | 랜딩 페이지 |
| `/how-it-works` | HowItWorksPage | 공통 | 이용 방법 안내 |
| `/login` | AuthPage | 공통 | 로그인 |
| `/signup` | AuthPage | 공통 | 회원가입 |
| `/mentors` | MentorListPage | 공통 | 멘토/과외 검색 |
| `/mentor/:id` | MentorDetailPage | 공통 | 멘토 프로필 상세 |
| `/tutorial/:tutorialId` | TutorialDetailPage | 공통 | 과외 상세 정보 |
| `/mentor/apply` | MentorApplicationPage | 멘티→멘토 | 멘토 신청 |
| `/mentor/dashboard` | MentorDashboardPage | 멘토 | 멘토 대시보드 |
| `/tutorial/create` | TutorialFormPage | 멘토 | 과외 등록 |
| `/tutorial/edit/:id` | TutorialFormPage | 멘토 | 과외 수정 |
| `/mypage` | MenteeDashboardPage | 멘티 | 멘티 마이페이지 |
| `/payment/:tutorialId` | PaymentPage | 멘티 | 결제 페이지 |
| `/payment/complete` | PaymentCompletePage | 멘티 | 결제 완료 |
| `/mypage/payments` | PaymentHistoryPage | 멘티 | 결제 내역 |
| `/qna` | QnAListPage | 공통 | Q&A 목록 |
| `/qna/create` | QuestionCreatePage | 멘티 | 질문 작성 |
| `/qna/:questionId` | QuestionDetailPage | 공통 | 질문 상세/답변 |

## 데이터 모델 (ERD 기반)

```
User (사용자)
  └── Mentor (멘토 정보)
        └── Tutorial (과외)
              └── Payment (결제) ──→ Ticket (수업권)
                                        └── Lesson (수업)
                                              └── Question (질문)
                                                    └── Answer (답변)
              └── Review (리뷰)
```

## 시나리오 1: 멘티의 과외 신청 및 결제

### 1.1 과외 탐색
```
[홈페이지 /]
    │
    ▼ "멘토 찾기" 클릭
[멘토 목록 /mentors]
    │ - 기술 스택 필터링 (React, Python 등)
    │ - 가격대 필터링
    │ - 평점순/리뷰순 정렬
    │
    ▼ 멘토 카드 클릭
[멘토 상세 /mentor/1]
    │ - 멘토 프로필 확인
    │ - 보유 기술 스택
    │ - 리뷰 목록
    │ - 등록된 과외 목록
    │
    ▼ 과외 카드 클릭
[과외 상세 /tutorial/1]
```

### 1.2 과외 상세 확인
```
[과외 상세 /tutorial/1]
    │
    ├── 과외 정보 확인
    │   - 제목, 설명, 커리큘럼
    │   - 회당 가격, 수업 시간
    │   - 난이도, 진행 방식
    │
    ├── 멘토 정보 확인
    │   - 프로필, 경력
    │   - 평점, 리뷰 수
    │
    ├── 예약 현황 확인 (사이드바)
    │   - 요일별 예약 가능 상태
    │   - 원활/조금 혼잡/혼잡/불가
    │
    └── 수업 후기 확인
        - 수강생 리뷰 목록
```

### 1.3 수업 신청 모달
```
[과외 상세 /tutorial/1]
    │
    ▼ "수업 신청하기" 버튼 클릭
[수업 신청 모달 (Dialog)]
    │
    ├── 선택한 수업 정보 표시
    │   - 과외 제목
    │   - 1:1 맞춤형 멘토링
    │   - 가격
    │
    ├── 날짜 선택 (캘린더)
    │   - 2024년 5월 15일 (수)
    │
    ├── 시간 선택 (09:00~20:00 그리드)
    │   - 선택됨 (파란색)
    │   - 예약 완료 (회색, 비활성)
    │   - 선택 가능 (기본)
    │
    ├── 멘토에게 전달할 메시지
    │   - 학습 목표, 요청사항 입력
    │   - 최대 500자
    │
    ▼ "신청 및 결제하기" 버튼 클릭
[결제 페이지 /payment/1]
```

### 1.4 결제 진행
```
[결제 페이지 /payment/1]
    │
    ├── 신청 수업 정보
    │   - 과외 제목, 멘토명
    │   - 회당 가격, 수업 시간
    │   - ★ 수업 횟수 선택 (1~12회)
    │
    ├── 희망 수업 일정
    │   - 날짜 선택 (date picker)
    │   - 시간 선택 (dropdown)
    │
    ├── 멘토에게 전할 말 (선택)
    │
    ├── 결제 수단 선택
    │   - 신용/체크카드
    │   - 실시간 계좌이체
    │   - 카카오페이
    │
    ├── 이용약관 동의
    │
    └── 최종 결제 금액 (사이드바)
        - 주문 금액 (₩50,000 × 4회)
        - 할인 금액
        - 총 결제 금액: ₩200,000
    │
    ▼ "결제 요청하기" 버튼 클릭
[PortOne 결제창]
    │ - 카드 정보 입력
    │ - 결제 승인
    │
    ▼ 결제 성공
[결제 완료 /payment/complete]
    │
    ├── 결제 완료 메시지
    ├── 결제 상세 내역
    │   - 수업명, 멘토
    │   - 결제 금액
    │   - 결제 일시
    │   - 주문 번호
    │
    └── 버튼
        - "수업 상세 보기" → /mypage
        - "홈으로 이동" → /
```

## 시나리오 2: 멘티의 수업 관리

### 2.1 마이페이지 확인
```
[멘티 마이페이지 /mypage]
    │
    ├── 프로필 섹션
    │   - 닉네임, 이메일

    │
    ├── 예정된 수업

    │   - 멘토 정보

    ├── 최근 질문

    │
    └── 결제 내역 바로가기
```

## 시나리오 3: 멘티의 Q&A 이용

```
[Q&A 목록 /qna]
    │
    ▼ "질문하기" 버튼 클릭

    │   - 진행 중인 수업 목록

    ├── 질문 제목 입력
    │

    │
```

```
[질문 상세 /qna/1]
```

## 핵심 기능 요약

### 멘티 기능
1. **과외 탐색**: 멘토/과외 검색, 필터링, 상세 보기
2. **수업 신청**: 날짜/시간 선택, 메시지 작성
3. **결제**: 횟수 선택, PortOne 결제, 티켓 발급
4. **수업 관리**: 예정 수업 확인, 티켓 잔여 확인
5. **Q&A**: 질문 작성, 답변 채택

### 멘토 기능
1. **과외 관리**: 과외 등록/수정
2. **수업 관리**: 요청 수락/거절, 수업 완료 처리
3. **Q&A 답변**: 질문 답변 등록
4. **수익 관리**: 결제 내역 확인
